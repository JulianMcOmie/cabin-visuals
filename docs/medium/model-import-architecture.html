3D Model Import — Architecture

"Borderline, but it's made out of Tame Impala's face" — a Model instrument that renders a user-uploaded glTF and behaves like every other instrument (params, ports, modulators, effects, export). The bytes problem is already solved once: copy the audio pattern verbatim, aimed at a models bucket.

1 · Two pipelines, one proven shape

2 · The Model instrument

AspectDesign
ParamsmodelRef (string, set via upload/picker UI — not a slider) · the standard family: baseSize, baseX/Y/Z, baseHue as a TINT (lerp material color toward hue, amount param) · spinSpeed (opt-in, 0 default — the Cube convention).
Ports / notesenergy/scale/hue ports like the basic shapes; note hits pulse scale via the shared breathe/energy pattern — a meme face that thumps on the kick with zero extra work.
NormalizationOn parse: center to bounding-box center, uniform-scale to a 1.6-unit box (the Cube's footprint) — so baseSize means the same thing for a 2cm and a 200m model.
Abilities/effectsNone bespoke; clone/transform/shader effects apply as to any object (radial-copied meme faces are exactly the point).
FallbackMissing/unloaded ref renders a placeholder wireframe box — never a crash, never an empty scene mystery.

3 · Decisions & non-goals

DecisionWhy
glTF/GLB only, 25 MB capThe one format with a first-class three.js loader and embedded textures (GLB). OBJ/FBX conversions happen outside the app. The cap keeps buckets and load times sane; raise later if it pinches.
modelAssets catalog beside audioClipsOptional document field (absent = empty) → no version bump; serialize/hydrate mirror the audioClips special-case.
Bytes pattern copied, not abstractedTwo similar 50-line modules beat one premature generic asset layer; if a THIRD asset type appears, extract then.
Static geometry v1glTF animations are wall-clock temptations — the pause invariant forbids playing them as-is. If wanted later: sample animation clips BY BEAT (clip.time = f(beat)), a deliberate follow-up.
Non-goalsModel marketplace/library UI beyond a per-project list · draco/meshopt compression · skeletal animation · per-mesh material editing.
The one-sentence mental model: a fourth basic shape whose geometry comes from a bucket — the audio-bytes seams copied name-for-name, the mesh normalized into the Cube's footprint, and every existing system (ports, effects, export) treats it as furniture it already knows.
