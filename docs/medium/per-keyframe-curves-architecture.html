Per-Keyframe Curves — Architecture

Each automation segment carries its own curve, DAW-node style. The engine change is two lines (the segment asks its left keyframe before falling back to the lane); the feature is really a piano-roll UI problem — which is why it's medium, not easy.

1 · Where the curve lives — the left keyframe owns its segment

2 · Layered on the lane mode, not replacing it

LayerRole after this feature
Lane interpolation (exists)Becomes the DEFAULT — every segment without its own curve uses it. The Curve control keeps working and now reads "default curve".
Keyframe curve? (new, optional)Overrides the default for its one segment. Absent everywhere = exactly today's behavior — old documents are automatically valid (no upgrade).
sampleLanePicks the mode per segment instead of once per call: mode = left.curve ?? laneMode. The ease() table is untouched.

3 · The UI — two tiers, ship the first

TierInteractionVerdict
1 · Right-click the segmentRight-click between two keyframes (or on the left keyframe) in an automation lane → curve menu (the 7 modes + "Use lane default"). Selected segments show a small glyph at the midpoint when overridden.Ship this. Menu machinery exists (NestedMenu); hit-testing a segment is y-band-between-keyframes math the editor already does for notes.
2 · Drag the midpoint to bendAbleton's gesture — drag a segment's middle up/down to morph between ease shapes.Defer. It implies a continuous curvature parameter, not 7 discrete modes — a different data model. Name it, don't build it.

4 · Decisions & non-goals

DecisionWhy
curve? on Note, used only by automation lanesKeyframes ARE notes (pitch encodes value); an optional field ordinary notes never set costs nothing and avoids a parallel keyframe type. Copy/paste and undo inherit it for free.
Left-keyframe ownershipDeletion/merge semantics fall out cleanly; matches user expectation from other DAWs.
Lane mode = default, not deadJulia's per-lane control stays the fast path ("make this whole lane exponential"); per-keyframe is the exception mechanism.
Non-goalsContinuous bend parameter (tier 2) · bezier handles · curves on ordinary MIDI notes (meaningless) · automation-value drawing tools.
The one-sentence mental model: an optional curve on the left keyframe overrides the lane's default for one segment — two lines in sampleLane, one context menu in the lane, and old projects don't even know the field exists.
